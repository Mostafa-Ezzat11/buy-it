# 🛒 Buy It

A complete **Flutter E-Commerce** application built with **Flutter** and **Firebase**, featuring role-based authentication for Users and Admins, real-time data synchronization, and a clean, maintainable project structure.

## ✨ Features

### 👤 User Features

- Secure Login & Registration
- Firebase Authentication
- Form Validation
- Product Categories
- Product Details
- Quantity Selection
- Shopping Cart
- Edit or Remove Cart Items
- Place Orders
- Profile Page
- Logout Confirmation Dialog

### 👨‍💼 Admin Features

- Admin Authentication
- Add Products
- Edit Products
- Delete Products
- View Customer Orders
- View Order Details
- Manage Product Inventory

### ⚡ Additional Features

- Real-time Updates using StreamBuilder
- Cloud Firestore Database
- Provider State Management
- Reusable Custom Widgets
- Organized Project Structure
- Responsive Navigation
- Error Handling using SnackBars

# 📂 Project Structure

```
lib
├── constants
├── helpers
├── models
├── screens
├── services
├── widgets
└── main.dart
```

The project is organized to improve readability, scalability, and maintainability by separating business logic, UI components, and services.

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/Mostafa-Ezzat11/buy-it.git
```

## Navigate to the project

```bash
cd buy-it
```

## Install dependencies

```bash
flutter pub get
```

## Run the application

```bash
flutter run
```
